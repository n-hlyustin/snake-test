# JAVAFX snake game

### Requirements
 - Download [JDK 11 or later](http://jdk.java.net/) for your operating system.
Make sure `JAVA_HOME` is properly set to the JDK installation directory.

 - Install [Gradle BuildTool](https://gradle.org/install/).

### Linux / Mac

Run the project:

    ./gradlew run

### Windows

Run the project:

    gradlew run
